# My Dotfiles

This repository contains my personal dotfiles, managed with [chezmoi](https://github.com/twpayne/chezmoi), for easy setup and synchronization across multiple machines.

## Overview

These dotfiles include configuration files for various tools and applications that I use on a daily basis, such as:

- Shell configuration (bash, zsh, etc.)
- Nvim editor 
- Git configurations
- System configurations (Hyprland)
- Custom scripts

## Prerequisites

- [chezmoi](https://github.com/twpayne/chezmoi) installed.
